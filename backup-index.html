Parallax Scrolling

Pretty slow Background

Background moves a good bit slower than the user scrolls. You just change the top position of the background for every pixel the page is scrolled. In this example the background is moving 0.3 times as fast as the scroll.

Multiple Backgrounds

This guy is using two different backgrounds. There's a div with one background and then another div nested within it that has it's own background.

The outer div is moving at the same speed as the basketball div (0.3*scroll) while the inner div with the "faster" shoe is moving at 0.6*scroll.

This text block is unaffected so it moves at the same speed that the user scrolls

Completely stationary background

This one uses multiple backgrounds also

but instead of nesting divs, each with their own background, this one uses CSS3 to assign multiple backgrounds to one div. In this case it's four backgrounds.

Then when the user scrolls you can move them at different rates
